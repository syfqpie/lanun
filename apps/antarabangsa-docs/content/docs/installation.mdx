---
title: Installation
---

Pasang package teras terlebih dahulu:

```bash
# using npm
npm install @lanun/antarabangsa

# using yarn
yarn add @lanun/antarabangsa

# using pnpm
pnpm add @lanun/antarabangsa
```

Package ini menyediakan utiliti teras untuk terjemahan dan logik routing. Ia boleh digunakan dalam mana-mana persekitaran React.

### Optional: Next.js Middleware

If you plan to use Next.js middleware for automatic locale detection, domain redirects, or path rewrites, install the Next.js integration:

```bash
# using npm
npm install @lanun/antarabangsa-next

# using yarn
yarn add @lanun/antarabangsa-next

# using pnpm
pnpm add @lanun/antarabangsa-next
```

Only required for Next.js projects that need middleware support. You still need `@lanun/antarabangsa` as the core dependency.
