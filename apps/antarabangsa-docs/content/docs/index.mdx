`@lanun/antarabangsa-react` and `@lanun/antarabangsa-next` are a pair of lightweight internationalization utilities designed for modern React and Next.js applications. Together, they provide a consistent way to handle routing, locale resolution, translations, and domain-aware redirects across both client and server environments.

The goal is simple: one routing logic, one translation model, shared everywhere.
